Ratings & History

Two tools to manage ratings. Use Recalculate after fixing history records. Use Full Reset to wipe everything and start fresh.

⟳ Recalculate Elos from History

Replays all recorded matches oldest→newest. Resets to seeded Elos first, then rebuilds civElo, civWins, civLosses, wins, losses from scratch. Run this any time ratings look wrong.

📥 Import from matches.json

Reads all matches from matches.json and replays every game to compute ratings. Use for first-time setup or after the scraper adds new matches.

⚠ Full Reset

Wipes all match history AND resets every player to seeded Elos. Cannot be undone. Only use to start completely fresh.
